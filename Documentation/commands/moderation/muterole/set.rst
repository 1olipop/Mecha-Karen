Set
===

This is were you can `set/update` your mute role!

Parameters
----------
**Role [Required]**

The new role to be set as your mute role

**Layout:**
::
	-Muterole Set <Role>

Permissions
-----------
**Bot**
::
	Embed Links

**Author**
::
	Manage Guild/Server

Cooldown
--------
10 Seconds - Per User

Aliases
-------
This command has no aliases yet!

Example Usage
-------------
::

	-Muterole Set Muted

1. Sets your servers mute-role to the role `Muted`.
