Set
===

This is were you can `set/update` your mute role!

Parameters
----------
**Role [Required]**

The new role to be set as your mute role

**Layout:**
::
	-Muterole Set <Role>

Permissions
-----------
**Bot**
::
	Embed Links

**Author**
::
	Manage Guild/Server

Cooldown
--------
10 Seconds - Per User

-------
This command has no aliases yet!
