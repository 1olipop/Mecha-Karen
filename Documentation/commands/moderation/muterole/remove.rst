Remove
======

Delete your servers mute role

Parameters
----------
**This command doesn't have any extra arguments**

**Layout:**
::
	-Muterole Remove

Permissions
-----------
**Bot**
::
	Embed Links

**Author**
::
	Manage Guild/Server

Cooldown
--------
10 Seconds - Per User

Aliases
-------
Delete

Example Usage
-------------
::

	-Muterole Remove

1. Deletes the mute role from the database
