Warns
=====

View the warns which are registers to a user

Parameters
----------
**Member [Required]**

The member whose warns are going to be viewed

**Page [Optional]**
A number to change from each page

.. Tip:: Each page has 10 warns

**Layout:**
::
	-Warns <Member> [Page="1"]

Permissions
-----------
**Bot**
::
	Embed Links

**Author**
::
	Manage Messages

Cooldown
--------
10 Seconds - Per User

Aliases
-------
Warnings
